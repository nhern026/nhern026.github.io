### ASR Transfer for Endangered Formosan Languages
Trained Wav2Vec2 XLSR-53 models and studied source-language selection using similarity metrics (ATDS, token overlap, subword overlap, Levenshtein distance). Found that unrelated languages with high ATDS improved WER by up to 68%.

**Tech:** Wav2Vec2 · HuggingFace · PyTorch · Austronesian NLP
