ASR for Endangered Formosan Languages

Dr. Emily Prud'hommeaux · NSF Formosan Bank Project

Trained Wav2Vec2 XLSR-53 on 10-hour datasets for Austronesian languages (Amis, Paiwan). Used ATDS, token overlap, subword overlap, and Levenshtein distance to identify optimal source-target pairs. Extended to distant languages (English, Hungarian, Swahili) — unrelated languages with high ATDS scores cut WER by up to 68%. Provides a strategic framework for source language selection in revitalization ASR.
